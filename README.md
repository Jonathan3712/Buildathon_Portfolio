## Deployment — Netlify via GitHub

This repo is connected to Netlify for automatic deployment.

**Every push to `main` triggers a new deploy automatically.**

To deploy manually or set up fresh:

1. Go to [app.netlify.com](https://app.netlify.com)
2. Click **Add new site → Import an existing project**
3. Select **GitHub** → find `buildathon-portfolio`
4. Build settings:
   - Build command: *(leave blank)*
   - Publish directory: `.` (root)
5. Click **Deploy site**
